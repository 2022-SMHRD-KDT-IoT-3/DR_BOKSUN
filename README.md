## 1. 프로젝트명
-시각장애인/고연령층/시력이 악화된 고객들을 위한, 실시간 음성 API 기반의 스마트 약 보관함.
-challange point : 음성 API

## 2. 주요기능
-버튼을 누르면 등록된 보관함의 정보를 음성으로 출력
-웹사이트에 등록된 복용시간에 맞춰 음성으로 알려주는 알람기능
-알람이 울리고 제품을 사용하지 않은 상태에서 제품을 누를 시 확인 음성을 출력
